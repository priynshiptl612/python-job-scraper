# 🚀 Python Job Scraper

A beginner-friendly **Web Scraping & Data Analysis Project** built using Python.  
This project automatically collects job listings from a website and stores them in a structured CSV file.

---

## 📌 Project Overview

This scraper extracts job-related information from the following website:

🔗 https://roadmap.sh/projects/job-listings-scraper

🔗 https://realpython.github.io/fake-jobs/

The collected data includes:

- 💼 Job Title
- 🏢 Company Name
- 📍 Location
- 🔗 Job Application Link

All extracted data is saved into a CSV file for easy analysis and usage.

---

# 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| Python | Main programming language |
| requests | Sends HTTP requests |
| BeautifulSoup | Parses HTML content |
| pandas | Stores and exports data |
